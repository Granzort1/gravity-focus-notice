# 更新のお知らせ

## 2026-04-22 アップデートのお知らせ

こんにちは、アプリ開発者です。

今回のアップデートでは、計画メニューを「1日のタイムスケジュール表」形式へと全面的に作り直しました。

## アップデート内容

1) 計画メニューを全面リニューアル — 1日のタイムスケジュール表
- これまでの計画 / 実行メニューの利用率が非常に低かったため、より実用的に「1日のタイムスケジュール表」を作るためのメニューとして計画メニューを新しく作り直しました。
- 下の例のように、1日を時間帯で区切り、各時間にやることを書き込んで1日全体を設計していくスタイルです。

### 1日の計画表の例

| 時間帯 | やること |
| --- | --- |
| 午前5〜6時 | 睡眠 |
| 午前6〜7時 | 6:30 起床、シャワー、着替え |
| 午前7〜8時 | 7:00〜7:30 朝食 / 7:30 犬の散歩 / 7:50までに家を出発 |
| 午前8〜9時 | 電車で出勤、オフィスへ移動 |
| 午前9〜10時 | 9:00〜9:10 1日の計画作成 / 9:10〜10:00 メール・電話 |
| 午前10〜11時 | 10:00〜10:30 会議 |
| 午前11〜12時 | 11:55〜12:00 やること追加 / エンジンオイル交換 |
| 正午〜午後1時 | 昼食、休憩、軽い散歩 |
| 午後1〜2時 | 13:00〜13:45 月次レポートの要約作成 / 13:45〜14:00 メール・電話 |
| 午後2〜3時 | 必要な業務の遂行 |
| 午後3〜4時 | 15:00〜15:30 会議 / 15:30〜16:00 議事録整理・予定表に記入 |
| 午後4〜5時 | 16:45〜17:00 メール・電話、明日の予定確認 |
| 午後5〜6時 | 電車で帰宅 |
| 午後6〜7時 | 18:00 クリーニング店に寄る、帰宅、犬の散歩、郵便物確認 |
| 午後7〜8時 | 19:00〜19:30 夕食 / 19:30〜20:00 家の掃除、自由時間 |
| 午後8〜9時 | 20:00〜20:30 自由時間(20:25 運動着に着替え) / 20:30〜21:00 ランニングマシン |
| 午後9〜10時 | シャワー、明日の服選び、余暇 |
| 午後10〜11時 | 余暇、23時までに就寝 |

- このように1日を時間帯ごとに配置しておくと、いつ何をするかが一目でわかるので、先延ばしや抜け漏れが減ります。
- この例の通りにする必要はありません。ご自身の生活リズムに合わせて、時間とやることを自由に書き込んでお使いください。

## ご意見をお聞かせください

新しい計画メニューについてご不便な点や改善のアイデアがございましたら、お気軽にお知らせください。設定メニュー上部のメールアイコン、または下部の「フィードバックを送る」からお送りいただけます。皆さまの一つ一つのご意見が、アプリをより良くする大きな力になります。

ありがとうございます。
