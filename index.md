# 업데이트 안내

## 2026-05-25 업데이트 안내

안녕하세요 앱 개발자입니다.

이번 업데이트에서는 일정 관리 기능을 강화하고, 일기·생각정리함을 일자별로 더 편리하게 살펴볼 수 있게 다듬었습니다. 윈도우 앱에는 포모도로 미니창과 바탕화면 캘린더 위젯이 새로 추가됐습니다.

## 업데이트 내역

1) "일정" 메뉴 추가 + "일일 계획" → "시간표" 메뉴명 변경
- 달력으로 기간과 일자별 일정을 기록할 수 있는 "일정" 메뉴가 새로 생겼습니다. 기존의 하루 단위 시간 계획 기능은 장기적인 일정 기록에는 적합하지 않아, 캘린더 기반의 일정 기록 메뉴를 별도로 만들었습니다.
- 함께, 기존 "일일 계획" 메뉴는 하루 안의 시간 계획이라는 점이 한눈에 전해지도록 "시간표"로 이름을 바꿨습니다.

2) 일기·생각정리함에 일자선택 달력 추가
- 두 메뉴에 날짜를 고를 수 있는 달력이 생겼습니다. 일자별로 적어둔 기록을 더 편하게 돌아보실 수 있습니다.

3) 윈도우 앱 신규 기능
- 포모도로 미니창: 포모도로를 시작한 뒤 앱을 최소화하면, 화면 한쪽에 작은 포모도로 창이 떠서 진행 상황을 보여드립니다. 평소에는 앱을 최소화해 두고 편하게 작업하시고, 남은 시간이나 상태는 미니창으로 가볍게 확인하세요.
- 바탕화면 캘린더 위젯: "일정" 메뉴의 달력을 윈도우 바탕화면에 고정 표시할 수 있습니다. 윈도우 앱의 *설정 → 바탕화면 캘린더 표시* 토글을 켜시면 활성화됩니다. 앞으로 일정 관리는 바탕화면 캘린더 위젯으로 편하게 다뤄보세요.
(윈도우 앱은 업데이트 심사에 시간이 좀 더 걸리고 있습니다 5월 25일 ~ 26일 사이에 업데이트 될 것 같습니다./ 업데이트 완료 <버전 1.0.53>)

앞으로는 사용자 편의성과 친화성 개선에 더 집중할 예정입니다. 불편하신 부분이나 개선 의견이 있으시면 언제든 skadldh@gmail.com 으로 메일을 보내주시거나, 설정 메뉴 상단의 쪽지 기능으로 말씀해 주세요. 사용자분들의 실사용 의견이 개선에 매우 큰 도움이 됩니다.

감사합니다.

## 2026-04-30 업데이트 안내

안녕하세요 앱 개발자입니다.

이번 업데이트에서는 포모도로 화면을 새로 디자인하고, 동기화 안정성을 추가로 잡았습니다.

## 업데이트 내역

1) 포모도로 화면 새 디자인
- 작업을 누르면 시간을 묻지 않고 바로 집중 모드로 들어갑니다.
- 화면 중앙에 호흡 오브가 표시되어 들숨과 날숨 리듬을 차분히 따라갈 수 있습니다.
- 일시정지 중에는 정지 버튼이 함께 표시되고, 우측 하단 체크 버튼으로 작업을 완료하면 다음에 진행할 작업을 바로 선택할 수 있습니다.
- 집중 도중 떠오르는 딴 생각은 화면 안 카드에 한 줄로 기록할 수 있습니다.
- 포모도로에서 작업을 완료해도 오늘 탭의 체크가 적용되지 않던 문제를 함께 수정했습니다.

2) 동기화 안정성 추가 보강
- Windows-Android 간 동기화 처리에서 일부 작업이 누락되거나 중복되던 케이스를 추가로 보완했습니다.
- 오래 지난 삭제 항목 정리 로직을 견고화하여, 시간이 지난 후 동기화에서 데이터가 어긋나지 않도록 했습니다.

감사합니다.
