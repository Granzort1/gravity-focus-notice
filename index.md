# 2026-04-10 업데이트 안내

안녕하세요 앱 개발자입니다.

이번 업데이트에서는 생각 정리함의 할 일 구분 기능을 개선하고, 기간 일정 추가 방식을 바꾸고, 첫 화면 진입 방식을 수정했습니다.

## 업데이트 내역

1) 생각 정리함 구분 기능 개선
- 할 일을 적을 때 쉼표나 줄바꿈으로 구분하면 자동으로 작업 목록으로 정리됩니다.
- 자유롭게 할 일을 적고, 오늘 할 일을 골라서 작업으로 만들어보세요.

2) 기간 일정 추가 버튼 변경
- 기간 일정 추가 버튼의 모양이 새롭게 바뀌었습니다.
- 실행 메뉴에서도 기간 일정을 추가할 수 있게 되어, 이제 계획과 실행 두 곳 모두에서 추가할 수 있습니다.

3) 첫 시작 화면 개선
- 앱을 처음 사용할 때 나오는 안내 화면을 개선했습니다.
- 바로 게스트 모드로 시작하는 대신, 게스트 모드와 Google 로그인 중 선택해서 시작할 수 있도록 변경했습니다.
